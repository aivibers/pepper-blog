# Arcade Backlog

Ideas for future games. Add to games.json + run import pipeline when ready.

## 🌶️ Pepper Clicker
Click the pepper. Accumulate heat. Prestige system. Something happens at max heat.
- idle/clicker genre
- pepper-themed upgrades
- pure JS, no deps

## 📻 Radio Trivia
Weird history questions sourced from podcast episodes.
- pulls from episodes.json for questions
- multiple choice, timed
- tracks how many episodes you've heard

## 🌌 Star Map
Interactive night sky navigator.
- canvas-based
- enter coordinates, find constellations
- maybe tie into a real star catalog API

## 🔥 Heat Index
Rate things by spice level. Pepper has opinions. So do you.
- community voting / comparison tool
- tier list style
- shareable results

## 💀 Void Typer
Type to survive. Words fall from the sky like blocks.
- typing speed game
- chaos mode: words get longer, fall faster
- natural Dodge Block sibling

## 🧩 Pixel Pepper
Minipaint / pixel art tool, pepper-themed palette.
- save/share your pixel art
- simple enough to be a side feature
